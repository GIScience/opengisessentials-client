Praktische Beispiele zu OpenGIS Web Map Service, Styled Layer Descriptor, Symbology Encoding, Filter Encoding und ISO 19117 Portrayal

 

Hier finden Sie praktische Beispiele zu dem Buch "OpenGIS Web Map Service, Styled Layer Descriptor, Symbology Encoding und ISO 19117 Portrayal vorgestellt und erläutert" von Christine Andrae, Christian Graul, Martin Over und Alexander Zipf.

 

 

Alle Beispiele, die Sie unterhalb des Menüpunktes "WMS-SLD Beispiele" finden, lassen sich online editieren. Es ist ebenso möglich eigene WMS Dienste einzubinden. Unter dem Menüpunkt "Hilfe" erfahren Sie mehr dazu.

Bitte beachten Sie, dass die hier verwendete Kartenserver Software den OGC Standard Styled Layer Descriptor 1.1.0 bisher nicht zu 100 % unterstützt. Es kann daher vorkommen, dass eigene Editierungen aus diesem Grunde nicht funktionieren, obwohl diese dem OGC Standard entsprechen.
